The $KINU Token

A BEP-20 compliant token with special features. There is a 12% tax on every buy and sell. That 12% tax goes to marketing, gaming pool and liquidity. The funds in the treasury wallet are used for marketing and further development of the $KINU ecosystem.

Token Task

For each transaction (transfer/buy/sell) there is a 12% Fee. The breakdown is as follows:

  • 6% allocated to marketing: These funds will be used to expand marketing to promote the project along with funding of the game and metaverse development.
  • 3% allocated to liquidity: This will allow for smooth transactions of the native $KINU Token and ensure no volatility when trading.
  • 3% allocated to the gaming pool: This percentage will fuel the growth of an interactive game pool, where community members will be able to win financially for being top players in the game.
